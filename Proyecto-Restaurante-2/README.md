# Sabores de Chile

Proyecto web desarrollado en HTML5, CSS3 y JavaScript para representar la página principal de un restaurante de gastronomía chilena. El sitio fue diseñado como una propuesta académica para la asignatura **Programación Web I**, incorporando estructura semántica, tablas, listas, enlaces, anclas, imágenes, audio y diseño adaptable.

## Estructura del proyecto

```text
sabores-de-chile/
│── index.html
│── README.md
│── css/
│   └── style.css
│── js/
│   └── script.js
│── img/
│   ├── pastel-choclo-1.png
│   └── pastel-choclo-2.png
│── audio/
│   └── pastel-choclo.mp3
```

## Características principales

- Página principal con sección de bienvenida o hero.
- Menú de navegación con anclas internas.
- Sección “Quiénes Somos”.
- Tablas para platos, bebidas y postres.
- Lista de definición para especialidades de la casa.
- Lista no ordenada para promociones.
- Sección “Plato de la Semana” con dos imágenes y un audio.
- Enlace externo informativo sobre el pastel de choclo.
- Diseño responsive para computador, tablet y móvil.
- Animaciones suaves de aparición y menú adaptable con JavaScript.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- Google Fonts (Poppins)

## Cómo ejecutar el proyecto

1. Descarga o descomprime la carpeta del proyecto.
2. Verifica que las carpetas `css`, `js`, `img` y `audio` estén dentro del mismo directorio principal.
3. Abre el archivo `index.html` en tu navegador.

## Recomendación para entrega académica

Para evidenciar la implementación, se sugiere:

- Incorporar el código fuente completo en el informe.
- Agregar capturas de pantalla del sitio funcionando en el navegador.
- Explicar brevemente el uso de encabezados, párrafos, tablas, listas, multimedia y anclas.
