# Portafolio de Carlos Di Piazza

Portafolio profesional con chatbot automatizado basado en CV.

## Tecnologías
- HTML5
- CSS3
- JavaScript
- Node.js
- Express

## Estructura
- `index.html`: página principal
- `css/estilo-index.css`: estilos
- `js/min.js`: animaciones del sitio
- `js/chatbot.js`: lógica del chatbot
- `server/server.js`: backend del chatbot
- `server/cv-data.json`: base de conocimiento del asistente

## Cómo ejecutar el proyecto

### Frontend
Abrir `index.html` en el navegador.

### Backend
```bash
cd server
npm install
npm start
